Chapter 6: Programming Languages

TERM DEFINITION
Machine Language The set of symbolic instruction codes usually in binary form that is used to represent operations and data in a machine.
Debugging The process of identifying and removing errors from computer hardware or software.
Assembler A program for converting instructions written in low-level symbolic code into machine code.
Assembly Language The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment.
Translator A programming language processor that converts a computer program from one language to another.
Compiler A computer program that translates computer code written in one programming language (the source language) into another language (the target language).
Interpreter A program that directly executes the instructions in a high-level language, without converting it into machine code.
Cross-Platform Software Computer software that is designed to work in several computing platforms.
Natural Language A form of artificial intelligence (AI) that allows computers to understand human language, whether it be written, spoken, or even scribbled.
Formal Language A language for defining the grammar of programming languages and formalized versions of subsets of natural languages in which the words of the language represent concepts that are associated with meanings or semantics.
Machine Independence Software that is not dependent on the properties of a particular machine, and can therefore be used on any machine.
High-Level Programming Language Any programming language that enables development of a program in a much more user-friendly programming context and is generally independent of the computer's hardware architecture.
Object-Oriented Programming(OOP) A computer programming model that organizes software design around data, or objects, rather than functions and logic.
Scripting Language A programming language that is used to manipulate, customize, and automate the facilities of an existing system.
Integer A data type used to represent real numbers that do not have fractional values.
Float A data type composed of a number that is not an integer, because it includes a fraction represented in decimal format.
String A data type used in programming, such as an integer and floating point unit, but is used to represent text rather than numbers.
Primitive Data Type A set of basic data types from which all other data types are constructed.
Data Structure A way of organizing and storing data in a computer so that it can be accessed and used efficiently.
Python Collection Data Structures A way of organizing data so that it can be accessed more efficiently depending upon the situation.
List Any information displayed or organized in a logical or linear formation.
Tuple A data structure that is an immutable, or unchangeable, ordered sequence of elements.
Set An abstract data type that can store certain values, without any particular order, and no repeated values.
Dictionary An unordered set that stores a value with some key and extracting the value given the key.
Function A sequence of program instructions that performs a specific task, packaged as a unit.
Local Variable A variable that is given local scope. A local variable reference in the function or block in which it is declared overrides the same variable name in the larger scope.
Global Variable A variable that is declared outside any function and is accessible to all routines in our program.
Source Program Instructions that guide the programs execution and the process of writing source code is commonly referred to as coding or programming.
Object Program A programming paradigm in computer science that relies on the concept of classes and objects.
